# UnlimitedWorldEditBridge
A WordEdit Addon similar to Fawe. Working on Paper 1.12.2 with Java 21
Made specially for Eagler.host 1.12 Servers

## Functions
- //walls
- //undo
- //set
- //copy
- //paste
- //redo
*Replaces WorldEdit Command

## Info:
- Dependencies: WorldEdit Bukkit 6.2.2
- Permissions: Based on WorldEdit permissions
- Other:
- - You can translate or change messages on messages.yml
- - There's nothing to see on config.yml
- - My server supported //set 6.7 MILLION blocks
